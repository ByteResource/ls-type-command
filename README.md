# Alternativa al comando ls (Linux) o dir (Windows)
Alternativa simple, con posibilidad de extenderse al comando ls o dir para enlistar archivos y directorio (carpetas).

### Capacidades
- Listar archivos y directorios (carpetas) de una ruta.
- Brindar información de dicho archivo o carpeta: nombre, peso, fecha de creación, etc...(opcional, con un argumento) por default dará el nombre.
- Ser multiplataforma. El programa deberá diferenciar al inicio el OS donde se está ejecutando (Windows o Linux) MacOS es basicamente linux asi que la primera version servirá.

### Versiones
- La primera solo listará los archivos y carpetas de la ruta asignada. (Por default, la ruta desde donde se ejecuta el programa).
- Añadir alguna diferenciación entre archivo y carpeta.
- Añadir la capacidad de leer argumentos y usar -i o -info para que enliste con toda la información del archivo.
- Brindar soporte para Windows.

## Alternative to ls (Linux) or dir (Windows) 
command Simple alternative, with the possibility of extending to ls or dir command to list files and directory (folders). 

### Capabilities 
- List files and directories (folders) from a path. 
- Provide information about said file or folder: name, weight, date of creation, etc... (optional, with an argument) by default will give the name. 
- Be multiplatform. The program must differentiate at the beginning the OS where it is running (Windows or Linux) MacOS is basically linux so the first version will work.

### Versions 

- The first will only list the files and folders in the assigned path. (By default, the path from which the program runs.) 
- Add some differentiation between file and folder. 
- Add the ability to read arguments and use -i or -info to list with all the information in the file.
